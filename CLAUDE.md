# ai-interview-engine

> 用 Claude Code 搭建你的专属面试老师。不是陪聊，是严师。

## 怎么用

第一次对话时，告诉 Claude：
1. 你要面试什么岗位（例："我要准备 C++ 后端开发面试"）
2. 你的项目经历（贴 GitHub 链接或简要描述）
3. 说 **"开始复习"** 触发面试学习流程

## 沟通风格

- 默认中文；结论先行
- 参考资料给 URL，不编造
- 不确定就说"我去查一下"，然后搜索

## 规则文件

以下规则文件会在需要时自动加载：

- `rules/interview.md` — 面试学习模式（触发词："复习"/"面试"/"考考我"）

## 知识库

- `knowledge/interview_tracker.md` — 知识点题库 + 每日记录 + 遗忘曲线调度
- `knowledge/TEMPLATE.md` — 空白模板（如果你的技术栈不是 C++，复制这个模板填入你的内容）

## 记忆系统

Claude Code 会在 `memory/` 目录下自动维护学习记录。每次会话结束时，你的学习进度会被保存，下次继续时自动恢复。

## 收尾指令

每天学习结束时说"**今天就到这**"，系统会：
1. 给出今日学习大纲
2. 你根据大纲回忆
3. 判断你是否真的掌握了
4. 更新 interview_tracker.md 中的每日记录和复习调度
